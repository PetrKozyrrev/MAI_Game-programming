## Козырев Пётр: М8О-412Б-22
# Программирование игр

## Описание игры
Главная цель - выжить как можно дольше, уничтожая врагов и маневрируя между препятствиями. Игра сочетает в себе элементы аркадного симулятора вождения и шутера. Важной особенностью является механика перегрева орудия, которая заставляет игрока вести стрельбу по врагам разумно.

## Реализованные механики
- Игровое поле с 4 разными локациями и препятствиями на них
- Управление автомобилем через WASD
- Камера плавно следует за игроком, всегда удерживая вид «из-за багажника» для лучшего обзора поля боя.
- Пушка на крыше автомобиля вращается независимо от направления движения, следуя за курсором мыши.
- Реализована шкала перегрева. При непрерывной стрельбе пушка временно блокируется, требуя времени на остывание.
- В игре представлено два типа врагов, использующих Unity NavMesh для навигации:
  - Ближний бой: Быстро преследует игрока и наносит урон при столкновении.
  - Стрелок: Умный противник, который держится на расстоянии и ведет огонь (стрельба с упреждением — враг рассчитывает точку выстрела, исходя из текущей скорости игрока и дистанции)
- Автоматическая генерация врагов в случайных точках по периметру арены.
- Система здоровья для игрока и врагов.
- Интерфейс со счетчиком уничтоженных врагов и полоской здоровья игрока. Автоматический перезапуск уровня при поражении.
